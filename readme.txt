4 steps to launch the bot:
1. Copy the "Telegram_bot" directory of the current version.

2. Must to install libraries:
    Go to "Telegram_bot" directory, write in console: pip install -r requirements.txt

3. Create an .env file in the "Telegram_bot" directory and paste your details.

Example .env file:
    TGAPI = "Your Telegram_api_token"
    GID = "Group_ID_(integer)"
    NDB = "your_path_to_database"

4. Run main.py
